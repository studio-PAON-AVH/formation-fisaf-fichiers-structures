# Formation FISAF – Fichiers Structurés

Présentation et ressources pour la formation FISAF sur la production de fichiers structurés pour la transcription et l'édition numérique adaptée.

## 🎯 Objectifs

- Comprendre les enjeux de l'édition numérique accessible
- Comprendre les possibilités offertes par les formats structurés
- Savoir convertir un fichier structuré simple
- Savoir produire un ouvrage simple au format EPUB3
- Savoir contrôler et évaluer un fichier `.epub`

## 📅 Programme

| Jour | Matin | Après-midi |
|------|-------|------------|
| **Jour 1** | Histoire de l'édition adaptée & XML | Produire et utiliser des fichiers XML DTBook |
| **Jour 2** | Le format EPUB | Produire des EPUBs accessibles |

## 🚀 Présentation en ligne

La présentation est générée avec [MkSlides](https://github.com/MartenBE/mkslides) et publiée sur GitHub Pages.

## 🛠️ Développement local

```bash
pip install mkslides
mkslides serve slides/
```
